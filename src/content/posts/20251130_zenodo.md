## 概要
当該コードは個人的な勉強目的で作成したものであり、分子構造操作や計算化学的処理を行うための機能を含んでいる。スパゲティコードであり修正は難しいかもしれないが、どうぞご自由に修正して使っていただきたい。

## 保存内容
- プログラム本体
- README / 使用方法のドキュメント
- 入力ファイルのサンプル（必要に応じて）

## 公開の目的
コードを公開することで、将来的な改良を促すことを目的とする。また、他者の発展的利用を歓迎する。

## 推奨引用方法
以下の形式で引用されたい。

```
ss0832. (2025). MultiOptPy: Multifunctional geometry optimization tools for quantum chemical calculations (v1.20.0). Zenodo. https://doi.org/10.5281/zenodo.17759573
```
